# UI Module
This directory will contain the UI logic extracted from `app.js`.

**Suggested Structure:**
- `dashboard.js` - Contains `showDashboard`, `loadDashboardData`, `applyDashboardRealtimeFilters`
- `history.js` - Contains `showHistory`, `exportFilteredPDF`, `applyFilters`
- `animations.js` - Contains GSAP timeline configurations
- `charts.js` - Contains Chart.js logic
